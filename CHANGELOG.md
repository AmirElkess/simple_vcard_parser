# Changelog

## [0.1.0] - 01-10-2020
* Initial release.

## [0.1.1] - 01-10-2020
* Minor fixes.

## [0.1.2] - 01-10-2020

## [0.1.3] - 06-10-2020

* Added support for gender property.

* telephone getter is deprecated, typedTelephone should be used instead.
* fixed a bug where lowercase tags are not detected.
